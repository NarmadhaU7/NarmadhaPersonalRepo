MVC - ModelViewController 
Model: It includes all the data and its related logic
View: Present data to the user or handles user interaction(UI)
Controller: An interface between Model and View components. (Includes Services which consist of functionalities)

From UI, the api request hits the controller. The controller(services) handles the request and perform operations in the db by making use of model.

view -> controller -> services (make use of model)-> repository -> db
               |-> gateway for API Requests.
